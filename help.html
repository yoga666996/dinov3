Frequently Asked Questions

DINOv3 can be installed in several ways:

  1. Via pip: pip install dinov3
  2. From source: Clone the repository and install dependencies
  3. Docker: Use our pre-built Docker containers

For detailed instructions, see our Installation Guide.

Minimum Requirements:

  • Python 3.8 or higher
  • PyTorch 1.12 or higher
  • 8GB RAM (16GB recommended)
  • CUDA-compatible GPU (optional but recommended)

Recommended: NVIDIA GPU with 8GB+ VRAM for optimal performance.

import torch
from dinov3 import DINOv3

# Load pre-trained model
model = DINOv3.from_pretrained('dinov3_vitb14')
model.eval()

# Process an image
with torch.no_grad():
    features = model(image_tensor)

Troubleshooting

Common Issues

CUDA Out of Memory

Reduce batch size or use gradient checkpointing to handle GPU memory limitations.

Learn More

Model Loading Errors

Ensure you have the correct PyTorch version and sufficient disk space for model downloads.

Learn More

Slow Inference

Optimize your model with TorchScript compilation or use mixed precision training.

Learn More

Performance Tips

  • Use GPU acceleration when available
  • Batch multiple images for better throughput
  • Consider model quantization for deployment
  • Cache preprocessed data for repeated use
  • Use appropriate image resolution for your task
